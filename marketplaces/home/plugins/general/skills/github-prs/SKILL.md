  Nutze wenn GitHub-Pull-Requests aufgelistet, geöffnet oder gereviewt werden sollen: eigene PRs + solche, wo
  man Reviewer ist; PRs mit Titel/Body aus dem Branch öffnen — über den offiziellen github-MCP. Merge/Review
  nach Bestätigung.
---

## Aktionen

- **list-open-prs:** meine PRs + PRs wo ich Reviewer bin
- **open-pr:** `gh pr create` — Branch → main, Labels, optionale Issue-Verknüpfung — **[CONFIRM]**
- **review-pr:** Review-Kommentar erstellen → reviewer-Plugin
- **ready:** Draft-PR als Ready markieren
