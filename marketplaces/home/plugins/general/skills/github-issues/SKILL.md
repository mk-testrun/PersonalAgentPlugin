## Aktionen

- **list-unassigned:** offene Issues ohne Assignee
- **list-mine:** mir zugewiesene Issues
- **list-bugs:** offene Bug-Issues
- **create-issue/bug:** mit Template + Labels — **[CONFIRM]**
- **create-task/close-task:** Task erstellen/schließen — **[CONFIRM]**
- **review-issue:** Verbesserungen vorschlagen (kein Auto-Write)
